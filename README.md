# Ennea Evolution — marketing site

Marketing site for **Ennea Evolution** (Candace), a counseling, coaching, and
organizational-development practice in Cache Valley, Utah.
Tagline: *Life evolves and so can we.*

Built fresh from the "Master Web Copy" with the brand book colors
(teal `#155F73`, marigold `#EF9C20`). Static, no commerce — this is the
public-facing site. (Separate from the earlier Next.js commerce app.)

## Stack
- **Astro 4** (static) · **Fraunces** + **Figtree** webfonts
- Deploys on **Netlify** (`npm run build` → `dist/`)
- Contact + free-download forms are **Netlify Forms** (work once deployed)

## Develop
```bash
npm install
npm run dev      # http://localhost:4330
npm run build    # static output to dist/
```
